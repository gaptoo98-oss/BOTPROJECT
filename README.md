# 🌍 Global Warming Telegram Bot

Простой и информативный Telegram-бот на Python, который рассказывает о проблемах глобального потепления. Бот использует текстовое меню и интерактивные кнопки для навигации.

## 🚀 Функционал

Бот предоставляет информацию по четырем ключевым темам:
*   **🌡️ Факты:** Случайные научные факты об изменении климата.
*   **🔥 Причины:** Почему происходит глобальное потепление.
*   **⚠️ Последствия:** К чему приводит изменение климата.
*   **💡 Что делать?:** Практические советы по эко-привычкам.

## 📦 Установка и запуск

1.  **Клонируйте репозиторий** (или просто создайте файл `bot.py` с кодом).
2.  **Установите необходимую библиотеку:**
    ```bash
    pip install pyTelegramBotAPI
    ```
3.  **Получите токен:**
    *   Напишите [@BotFather](https://t.me) в Telegram.
    *   Создайте нового бота и скопируйте API Token.
4.  **Настройте код:**
    *   Откройте файл и вставьте ваш токен в переменную:
    ```
    TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
    ```
5.  **Запустите бота:**
    ```bash
    python bot.py
    ```

## 🎮 Использование
*   После запуска отправьте боту команду `/start`.
*   Используйте основную клавиатуру (внизу экрана) или Inline-кнопки (под сообщениями) для получения информации.

## 📂 Структура кода
*   `main_keyboard()` — создание обычных кнопок управления.
*   `inline_buttons()` — создание интерактивных кнопок под сообщениями.
*   `handle_inline()` — обработка нажатий на inline-кнопки.
*   `bot.infinity_polling()` — метод для бесперебойной работы бота.
